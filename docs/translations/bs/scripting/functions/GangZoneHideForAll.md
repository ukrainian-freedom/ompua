## Deskripcija

GangZoneHideForAll sakriva gangzonu za sve igrače.

| Ime  | Deskripcija     |
| ---- | --------------- |
| zone | Zone za skriti. |

## Returns

Ova funkcija ne returna (vraća) nikakve posebne vrijednosti.

## Primjeri

```c
new gGangZoneId;
gGangZoneId = GangZoneCreate(1248.011, 2072.804, 1439.348, 2204.319);
GangZoneHideForAll(gGangZoneId);
```
