:::

## Deskripcija

Dobiva najveći id vozila koji se trenutno koristi na serveru.

## Primjeri

```c
RepairAllVehicles()
{
    // vehicleids počinje sa 1
    for(new i = 1, j = GetVehiclePoolSize(); i <= j; i++)
    {
        RepairVehicle(i);
    }
}
```

## Srodne Funkcije

- [GetPlayerPoolSize](GetPlayerPoolSize): Dobija najviši playerid povezan na server.
