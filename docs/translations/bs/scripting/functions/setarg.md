## Deskripcija

Postavite argument koji je proslijeđen funkciji.

| Ime   | Deskripcija                                          |
| ----- | ---------------------------------------------------- |
| arg   | Broj sekvence argumenta. Koristi 0 za prvi argument. |
| index | Index (ako je argument niz/array).                   |
| value | Vrijednost na koju treba postaviti argument.         |
