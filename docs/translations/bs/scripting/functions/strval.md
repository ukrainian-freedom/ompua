## Deskripcija

Pretvori string u cijeli broj.

| Ime            | Deskripcija                                   |
| -------------- | --------------------------------------------- |
| const string[] | String kojeg želite pretvoriti u cijeli broj. |

## Returns

Cjelobrojna vrijednost stringa. '0 ako string nije numerički.

## Primjeri

```c
new string[4] = "250";
new iValue = strval(string); // iValue je sada '250'
```
