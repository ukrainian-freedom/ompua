---
title: Vehicle Health
description: Vehicle Health Values
---

| Health  | Engine Status                        |
| ------- | ------------------------------------ |
| > 650   | Undamaged                            |
| 650-550 | White Smoke                          |
| 550-390 | Grey Smoke                           |
| 390-250 | Black Smoke                          |
| < 250   | On fire (will explode seconds later) |

- [GetVehicleHealth](../functions/GetVehicleHealth): Get a vehicle's health.
