## คำอธิบาย

Get the number of arguments passed to a function.

| Name | Description |
| ---- | ----------- |


## ตัวอย่าง

```c
SomeFunction(...)
{
    printf("numargs(): %i", numargs());
}

public OnFilterScriptInit()
{
    SomeFunction(1, 2, 3);
}

// Output: "numargs(): 3"
// Because 3 parameters (1, 2, 3) were passed.
```
