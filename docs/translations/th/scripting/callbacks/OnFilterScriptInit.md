## คำอธิบาย

Callback นี้ถูกเรียกเมื่อฟิลเตอร์สคริปต์เริ่มต้นทำงาน (โหลด) มันถูกเรียกเฉพาะในฟิลเตอร์สคริปต์ที่กำลังเริ่มต้นทำงาน

| ชื่อ | คำอธิบาย |
| ---- | -------- |


## ตัวอย่าง

```c
public OnFilterScriptInit()
{
    print("\n--------------------------------------");
    print("Filterscript ได้เริ่มทำงานแล้ว");
    print("--------------------------------------\n");
    return 1;
}
```
