## Deskripsi

Mendapatkan panjang dari sebuah string.

| Nama           | Deskripsi                             |
| -------------- | ------------------------------------- |
| const string[] | String yang akan dihitung panjangnya. |

## Returns

Panjang string dalam bentuk integer.

## Contoh

```c
new stringLength = strlen("This is an example string."); // stringLength menjadi 26
```
