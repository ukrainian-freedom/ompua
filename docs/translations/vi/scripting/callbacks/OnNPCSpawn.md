## Mô Tả

Callback này được gọi khi một NPC được sinh ra.

## Ví Dụ

```c
public OnNPCSpawn()
{
    print("NPC đã được sinh ra");
    SendChat("Chào thế giới. Tôi là một bot.");
    return 1;
}
```
